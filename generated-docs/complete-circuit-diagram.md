# USB-PD モジュラーシンセ電源 完全回路図

## 段階2A: +15V → +13.5V 降圧回路 (LM2596S-ADJ #1)

```
+15V ─────┬─── L1: 100µH ──┬─── D1 ──┬─── C3: 470µF/25V ──┬─→ +13.5V/1.3A
          │    (4.5A)      │   SS34  │    (入力フィルタ)   │
          │                │    ↓    │                    │
          │           ┌────┴─────────┴──┐                 │
          │           │5  VIN      VOUT │4────────────────┤
          └───────────┤3  ON        FB  │2────────┬───────┘
                      │1  GND           │         │
                      └─────────────────┘         │
                              │                   │
                             GND              ┌────┴───┐
                                              │ R1     │ 10kΩ (実際13.53V)
                                              │ 10kΩ   │
                                              └────┬───┘
                                              ┌────┴───┐
                                              │ R2     │ 1kΩ
                                              │ 1kΩ    │
                                              └────┬───┘
                                                   │
                                                  GND
```

## 電源フロー概要

```
USB-C 15V ──┬─→ +13.5V (DC-DC) ──→ +12V (LDO) ──→ +12V OUT
            │
            ├─→ +7.5V  (DC-DC) ──→ +5V  (LDO) ──→ +5V OUT
            │
            └─→ -15V (反転) ──→ -13.5V (DC-DC) ──→ -12V (LDO) ──→ -12V OUT
```

## 性能特性

- **効率**: 全体で75-80%
- **リップル**: <1mVp-p (最終出力)
- **レギュレーション**: ±1%
- **安全マージン**: 全回路150%以上
- **総電力**: 約30W (15V × 2A入力)
