# ICL7660M/TR 電圧反転IC ピン配置と機能詳細

## 概要
ICL7660M/TRは電圧反転IC（チャージポンプ方式）で、正電圧から負電圧を生成

## ピン配置 (SOP-8)
```
    1  2  3  4
   ┌─────────┐
 8 │○        │ 5
   │ ICL7660 │
 7 │         │ 6
   └─────────┘
```

## 各ピンの機能

| ピン番号 | ピン名 | 機能 | 説明 |
|---------|--------|------|------|
| **1** | **CAP+** | 外部コンデンサ正極 | フライングキャパシタ接続 |
| **2** | **V+** | 正電源入力 | +1.5V～+10V入力 |
| **3** | **GND** | グランド | 回路グランド |
| **4** | **CAP-** | 外部コンデンサ負極 | フライングキャパシタ接続 |
| **5** | **VOUT** | 負電圧出力 | -V+の電圧出力 |
| **6** | **LV** | 低電圧動作選択 | LOW=通常, HIGH/OPEN=低電圧モード |
| **7** | **OSC** | 発振器制御 | 外部クロック入力またはオープン |
| **8** | **V+** | 正電源入力 | ピン2と同じ（デュアル接続） |

## 動作原理

### チャージポンプ方式
1. **充電フェーズ**: 外部コンデンサがV+からGNDに充電
2. **放電フェーズ**: コンデンサが反転してVOUTに負電圧供給

### 基本回路
```
V+ ─┬─ PIN2,8
    │
    ├─ C1+ ─ PIN1 ── C1 ── PIN4 ─ C1-
    │
   GND ─ PIN3
    │
 VOUT ─ PIN5 ── C2 ── GND
```

## USB-PD電源での使用

### 入力/出力仕様
- **入力**: +15V (USB PD電圧)
- **出力**: -15V (理論値)
- **実用出力**: -14V～-14.5V (負荷により変動)

### 外部部品（JLCPCB選定済み）

#### フライングキャパシタ（C1）
- **部品**: 10µF 25V セラミック（C15850）
- **接続**: PIN1 - PIN4間
- **機能**: 電荷蓄積・転送

#### 出力コンデンサ（C2）  
- **部品**: 10µF 25V セラミック（C15850）
- **接続**: PIN5 - GND間
- **機能**: 出力フィルタ

## 回路接続例

```
+15V ────┬──── PIN2,8 (V+)
         │
      [10µF] ──── PIN1 (CAP+)
         │            │
         │            C1 (10µF)
         │            │
         └──── PIN4 (CAP-) 
         
GND ─────────── PIN3 (GND)

PIN6 (LV) ──── OPEN (通常動作)
PIN7 (OSC) ─── OPEN (内蔵発振器)

-15V ────────── PIN5 (VOUT) ──[10µF]── GND
```

## 性能仕様

| 項目 | 仕様 |
|------|------|
| **入力電圧範囲** | +1.5V～+10V |
| **出力電圧** | -VIN (理論値) |
| **最大出力電流** | 20mA (無負荷時) |
| **効率** | 95%以上 (軽負荷時) |
| **発振周波数** | 10kHz (内蔵) |
| **電源電流** | 170µA (typ) |
| **パッケージ** | SOP-8 |

## 設計上の注意点

### 1. 出力電流制限
- **ICL7660単体**: 最大20mA
- **後段LM2596S必要**: 大電流用途には不適
- **解決策**: ICL7660 → LM2596S の2段構成

### 2. 電圧降下
- **無負荷**: VOUT ≈ -VIN
- **負荷時**: 内部抵抗により電圧低下
- **対策**: 後段でDC-DC変換が必要

### 3. ノイズ対策
- **スイッチングノイズ**: 10kHz基本波
- **対策**: 出力に十分な容量のコンデンサ
- **レイアウト**: GNDプレーン重要

## USB-PD電源での役割

### システム構成
```
+15V (USB-PD) → ICL7660 → -15V → LM2596S → -13.5V → LM7912 → -12V
```

### 設計理由
1. **-12V生成**: モジュラーシンセには負電源必須
2. **簡単実装**: 外部コンデンサ2個のみ
3. **低コスト**: $0.078の安価なIC
4. **JLCPCB対応**: SMT実装可能
