# LM2596S-ADJ ピン配置と機能詳細

## 概要
LM2596S-ADJは可変出力3A降圧DC-DCコンバータ（TO-263-5パッケージ）

## ピン配置 (TO-263-5)
```
     1    2    3
   [VIN][ON/OFF][GND]
   
     4    5
   [FB][VOUT]
```

## 各ピンの機能

| ピン番号 | ピン名 | 機能 | 説明 |
|---------|--------|------|------|
| **1** | **VIN** | 電源入力 | 4.5V～40V入力電圧 |
| **2** | **ON/OFF** | 制御入力 | HIGH=ON, LOW/GND=OFF, 内部プルアップあり |
| **3** | **GND** | グランド | 電源グランド、ヒートシンク接続 |
| **4** | **FB** | フィードバック | 出力電圧検出用（基準電圧1.23V） |
| **5** | **VOUT** | スイッチング出力 | 外部にインダクタ+ダイオード必要 |

## 出力電圧設定（FB ピン）

FB ピンの電圧分割で出力電圧を設定：
```
VOUT = 1.23V × (1 + R1/R2)
```

- **R1**: FB～VOUT間の抵抗
- **R2**: FB～GND間の抵抗

## USB-PD電源での使用例

### +13.5V出力設定（+12V用）
- **R1** = 10kΩ (FB～VOUT)  
- **R2** = 1kΩ (FB～GND)
- **計算**: VOUT = 1.23V × (1 + 10k/1k) = **13.53V**

### +7.5V出力設定（+5V用）
- **R1** = 5.1kΩ (FB～VOUT)
- **R2** = 1kΩ (FB～GND)  
- **計算**: VOUT = 1.23V × (1 + 5.1k/1k) = **7.51V**

### -13.5V出力設定（-12V用）
- **R1** = 10kΩ (FB～VOUT)
- **R2** = 1kΩ (FB～GND)
- **入力**: ICL7660で生成した-15V
- **計算**: VOUT = 1.23V × (1 + 10k/1k) = **-13.53V**

## 必要な外部回路

### インダクタ
- **L1,L2,L3**: CYA1265-100UH (100µH 4.5A)
- **接続**: VOUTピン → インダクタ → 出力

### ダイオード
- **D1,D2,D3**: SS34 (3A 40V ショットキー)
- **接続**: VOUTピン → カソード, アノード → GND
- **機能**: フリーホイールダイオード

### コンデンサ
- **入力**: 10µF 25V (VIN-GND間)
- **出力**: 470µF 電解コンデンサ (負荷後)

## 動作仕様

| 項目 | 仕様 |
|------|------|
| **入力電圧** | 4.5V～40V |
| **出力電流** | 最大3A |
| **効率** | 最大85% |
| **スイッチング周波数** | 150kHz |
| **温度範囲** | -40°C～85°C |
| **パッケージ** | TO-263-5 (ヒートシンク取付可能) |

## 設計上の注意点

1. **ヒートシンク**: 大電流時は必須（GNDピンがヒートシンク接続）
2. **レイアウト**: インダクタとダイオードは近接配置
3. **グランドプレーン**: 安定動作のため重要
4. **入力フィルタ**: スイッチングノイズ対策
