## Diagram4: +15V → -15V 電圧反転回路 (ICL7660)

```
+15V ─────┬─────────────────────────────────────────┬─→ +15V (他回路へ)
          │                                         │
          │           ICL7660M/TR                   │
          │         ┌──────────────┐                │
          └─────────┤2,8  V+   CAP+│1──┐            │
                    │              │   │            │
           GND ─────┤3   GND   CAP-│4──┼─┐          │
                    │              │   │ │          │
          OPEN ─────┤6   LV    V-  │5──┼─┼──────────┴─→ -15V
                    │              │   │ │
          OPEN ─────┤7   OSC       │   │ │
                    └──────────────┘   │ │
                                       │ │
                     C9: 10µF セラミック │ │
                    ┌───────────────────┘ │
                    │                     │
                    └─────────────────────┘

                    C10: 10µF セラミック
                    ┌─────────────────────────┐
                    │                         │
            -15V ───┼─────────────────────────┼─── GND
                    │                         │
                    └─────────────────────────┘
```

**フライングキャパシタ（C9）はPIN1とPIN4の間に直接接続**

```
      フライングキャパシタ C9 (10µF)
           ┌─────────┐
PIN1 ──────┤+       -├────── PIN4
(CAP+)     └─────────┘     (CAP-)
```

## Diagram9: 保護回路（初心者対応・2段階保護）

```
+12V ──┬─── PTC1: 1.1A ──┬─── F1: 2A ───┬─── TVS1: SMAJ15A ───┬─→ +12V OUT
       │   (自動復帰)     │  (バックアップ)│    (15V一方向)      │
       │                 │              │      ↓              │
       │                 │              └─────GND─────────────┤
       │                 │                                    │
       │                 └─── LED2: 緑 ──[R7: 1kΩ]──┬─→ 電源状態表示
       │                                             │
       │                                            GND
       │
       └─── 段階的保護:
            1. 過負荷 (1.2-2A) → PTC遮断 → 30秒後自動復帰 ✅
            2. 短絡 (>2A) → ヒューズ溶断 → 修理必要 ❌

+5V ───┬─── PTC2: 0.75A ─┬─── F2: 1.5A ──┬─── TVS2: PRTR5V0U2X ─┬─→ +5V OUT
       │   (自動復帰)     │  (バックアップ)│    (5V双方向)        │
       │                 │              │      ↕               │
       │                 │              └─────GND──────────────┤
       │                 │                                    │
       │                 └─── LED3: 青 ──[R8: 1kΩ]──┬─→ 電源状態表示
       │                                             │
       │                                            GND
       │
       └─── 段階的保護 (同上)

-12V ──┬─── PTC3: 0.9A ──┬─── F3: 1.5A ──┬─── TVS3: SMAJ15A ───┬─→ -12V OUT
       │   (自動復帰)     │  (バックアップ)│   (15V一方向逆接続)  │
       │                 │              │      ↑              │
       │                 │              └─────GND─────────────┤
       │                 │                                    │
       │                 └─── LED4: 赤 ──[R9: 1kΩ]──┬─→ 電源状態表示
       │                                             │
       │                                            GND
       │
       └─── 段階的保護 (同上)

※ TVS3は負電圧保護のため、カソードを-12Vライン、アノードをGNDに接続
※ PTCは通常時ほぼ抵抗ゼロ、過電流時に高抵抗化して電流を制限
※ ヒューズはPTCが対応できない急激な短絡から保護
※ LEDが消えたら過負荷 → モジュールを減らして30秒待つ
```

## Diagram10: 電源フロー概要

```
USB-C 15V ──┬─→ +13.5V (DC-DC) ──→ +12V (LDO) ──→ +12V OUT
            │
            ├─→ +7.5V  (DC-DC) ──→ +5V  (LDO) ──→ +5V OUT
            │
            └─→ -15V (反転) ──→ -13.5V (DC-DC) ──→ -12V (LDO) ──→ -12V OUT
```
