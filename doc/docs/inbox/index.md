## プロジェクトの特徴

### 4段階設計

1. **USB-PD給電部** (CH224Q)
2. **DC-DCコンバータ** (LM2596S × 3 + ICL7660)
3. **リニアレギュレータ** (LM7812/7805/7912)
4. **保護回路** (PTC + ヒューズ + TVS)

### 性能

- **出力**: +12V/1.2A, -12V/1A, +5V/1.2A
- **リップル**: \<1mVp-p
- **効率**: 75-80%
- **保護**: 2段階（PTC自動復帰 + ヒューズバックアップ）
