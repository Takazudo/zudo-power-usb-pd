# How to use tools

Technical guides and workflow documentation for the USB-PD Power Supply project.

## Documentation Workflows

- **[KiCad Footprint Generation](./kicad-footprint-generation.md)** - Download and manage KiCad footprints from LCSC/EasyEDA
- **[Create Footprint SVG Files](./create-footprint-svg.md)** - Export and prepare footprint images for documentation
- **[Create Circuit SVG Files](./create-circuit-svg.md)** - Generate professional circuit diagrams using schemdraw

## Purpose

These guides document the workflows used to maintain and update this project's documentation. They ensure consistency and make it easy to add new components or update circuit diagrams.
