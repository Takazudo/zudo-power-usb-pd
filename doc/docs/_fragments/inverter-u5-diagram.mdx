**Key Points:**
- **Charge pump operation**: ICL7660 uses C12 as a flying capacitor to generate negative voltage
- **Flying capacitor C12**: 10µF ceramic capacitor between CAP+ (pin 2) and CAP- (pin 4)
- **Output filter C13**: 10µF ceramic capacitor from VOUT to GND smooths switching ripple
- **Output voltage**: Typically -14V (nominal formula: VOUT ≈ -(VIN - 1V))
- **Oscillator**: Internal 10kHz oscillator (pin 7 OSC left open)
- **Low voltage mode**: Pin 6 (LV) left open for normal voltage operation
- **No external components needed**: Only two capacitors required for full operation
