**Key Points:**

- **Flyback topology**: LM2586 configured for negative voltage generation from positive input using coupled transformer
- **High current capability**: 3A output capability (easily handles 800mA requirement for -12V rail)
- **Input voltage range**: 4V~40V (safely accepts +15V input)
- **Output voltage**: Adjustable -15V via feedback resistor divider (R7/R8)
- **Switching frequency**: 200kHz (requires appropriate inductor selection)
- **Coupled transformer**: T1 with 1:1 turns ratio (47µH primary, 47µH secondary magnetically coupled)
- **Component count**: Transformer, Schottky diode, feedback network, and compensation network for stable high-current operation

**Component Selection:**

- **T1**: 47µH:47µH coupled flyback transformer (1:1 turns ratio) rated for 3A primary current
- **C13**: 100µF input bulk capacitor (electrolytic, rated for 25V+)
- **C16**: 100nF ceramic decoupling capacitor (X7R, 50V, 0805) - High-frequency filtering
- **C14**: 100µF output capacitor (electrolytic, reversed polarity for negative rail, rated for 25V+)
- **D4**: Schottky diode rated for 3A (e.g., SS34, MBRS340)
- **R7**: 10kΩ feedback resistor (1/4W, 1% tolerance)
- **R8**: ~920Ω feedback resistor (1/4W, 1% tolerance) for -15V output
- **R9**: 3kΩ compensation resistor (1/4W, 1% tolerance)
- **C15**: 0.047µF (47nF) ceramic compensation capacitor (X7R or C0G/NP0)
