**Key Points:**
- **Inverted SEPIC topology**: LM2586 configured for negative voltage generation from positive input
- **High current capability**: 3A output capability (easily handles 800mA requirement for -12V rail)
- **Input voltage range**: 4V~40V (safely accepts +15V input)
- **Output voltage**: Adjustable -15V via feedback resistor divider (R1/R2)
- **Switching frequency**: 200kHz (requires appropriate inductor selection)
- **Coupled inductor**: L1/L2 can use single coupled inductor or two separate inductors
- **Component count**: More complex than charge pump (2 inductors, coupling cap, diode, feedback network) but necessary for high current

**Component Selection:**
- **L1, L2**: 47µH inductors rated for 3A (or single 47µH coupled inductor)
- **C1**: 47µF ceramic or electrolytic coupling capacitor
- **CIN**: 100µF input capacitor
- **COUT**: 100µF output capacitor (electrolytic, reversed polarity for negative rail)
- **D1**: Schottky diode rated for 3A (e.g., SS34, MBRS340)
- **R1, R2**: Feedback resistor divider for -15V output
