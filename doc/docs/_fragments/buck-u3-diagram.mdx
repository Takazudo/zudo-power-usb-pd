```
                                          |
                     LM2596S-ADJ (U3)       |
                     +---------------+       |
+15V ---+------------+5 VIN          |       |
        |            |               |       |
        |     +------+3 ON           |       |
        |     |      |               |       |
        |     |      |          VOUT +4------+-> L2 ---+-> +7.5V/0.6A Output
        |     |      |                      100uH    |
        |     |      |            FB +2-> Tap        +-> R3 --+-> Tap -> R4 -> GND
        |     |      |                               | 5.1kΩ  |         1kΩ
        |     |      |           GND +1-> GND        |        |
        |     |      +---------------+               +-> C4 -> GND
        |     |                                         470uF
       C7    C8                                          |
      100uF 100nF                                       GND
        |     |                                      Schottky
       GND   GND                                     D2 (SS34)

Note: FB pin 2 connects to the tap point between R3 and R4.
      This tap voltage = VOUT × R4/(R3+R4) = 7.5V × 1kΩ/6.1kΩ = 1.23V
      The LM2596 adjusts duty cycle to maintain FB pin at 1.23V reference.

Connections:
  Input power:    +15V -> C7 (100uF) -> GND  (input filter, parallel)
                  +15V -> U3 pin 5 (VIN)
  Decoupling:     VIN -> C8 (100nF) -> GND  (high-freq decoupling, parallel)
  Enable:         ON pin 3 -> tied high (always enabled)

  Output stage:   VOUT (pin 4) -> L2 (100uH inductor) -> +7.5V Output
  Flyback diode:  Switching node (between pin 4 and L2) -> D2 (SS34 Schottky) -> GND
  Output filter:  Output -> C4 (470uF) -> GND  (parallel, NOT in series!)

  Voltage divider (sets output voltage):
                  Output -> R3 (5.1kΩ) -> Tap point -> R4 (1kΩ) -> GND
  Feedback:       FB pin 2 -> Tap point (junction between R3 and R4)
                  Tap voltage = 1.23V when output = 7.5V

  Ground:         GND pin 1 -> System ground (shared with all components)
```
