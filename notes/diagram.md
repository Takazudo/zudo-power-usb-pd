## Diagram9: 保護回路

```
+12V ──┬─── F1: 1A ───┬─── TVS1: SMAJ15A ───┬─→ +12V OUT (保護済み)
       │    (ヒューズ) │    (15V一方向)      │
       │              │      ↓              │
       │              └─────GND─────────────┘
       │
       └─── 過電流保護と電圧クランプ

+5V ───┬─── F2: 1A ───┬─── TVS2: PRTR5V0U2X ─┬─→ +5V OUT (保護済み)
       │    (ヒューズ) │    (5V双方向)        │
       │              │      ↕               │
       │              └─────GND──────────────┘
       │
       └─── 過電流保護と電圧クランプ

-12V ──┬─── F3: 1A ───┬─── TVS3: SMAJ15A ───┬─→ -12V OUT (保護済み)
       │    (ヒューズ) │    (15V一方向逆接続) │
       │              │      ↑              │
       │              └─────GND─────────────┘
       │
       └─── 過電流保護と電圧クランプ

※ TVS3は負電圧保護のため、カソードを-12Vライン、アノードをGNDに接続
```
